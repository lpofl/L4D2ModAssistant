# L4D2 模组管理器 (L4D2 Mod Assistant)

L4D2 MOD 管理助手 LMA 旨在帮助玩家高效存储管理他们的MOD，提供分类管理、tag管理、评分管理、冲突前后置关系记录，免去文件夹管理带来的繁琐、不便。同时提供按指定规则随机生成mod组合方案并批量导入游戏目录addons文件夹功能，使玩家随时切换所用mod，不用对收藏的几十G MOD选择困难，倒腾库存还嫌麻烦的苦恼

当前完成度有限，建议过半个月再来看

## 功能特性

- **模组导入:** 从本地文件夹轻松导入模组。
- **高级整理:** 使用自定义分类和标签来组织模组。
- **模组选择器:** 在一个集中的视图中浏览、筛选和管理您的模组仓库。
- **模组方案:** 保存和加载不同的模组配置方案。
- **随机化功能:** 根据您的标准随机选择模组。
- **模组编辑器:** 编辑每个模组的详细信息和属性。

## 技术栈

- **编程语言:** C++
- **图形界面框架:** Qt 6
- **构建系统:** CMake
- **包管理器:** vcpkg
- **数据库:** SQLite (用于存储模组信息)

## 项目结构

- `/app`: 包含应用程序的主要逻辑，包括 UI (View) 和 Presenter 层。
    - `/app/ui`: 基于 Qt 的 UI 组件 (窗口、页面、对话框)。
    - `/app/services`: 应用层级的服务。
    - `/app/ui/presenters`: 连接 UI 和核心逻辑的 Presenter 层。
- `/core`: 核心业务逻辑 (Model)，包括数据访问对象 (DAO)、服务和数据库管理。
- `/database`: 存放数据库文件。
- `/setting_config`: 包含初始化配置文件 (例如，默认的标签和分类)。
- `/tests`: 项目的单元测试。

### 环境要求

- C++ 编译器 (MSVC, GCC, 或 Clang)
- [CMake](https://cmake.org/download/) (3.15 或更高版本)
- [Qt](https://www.qt.io/download) (6.x 版本)
- [vcpkg](https://github.com/microsoft/vcpkg)
